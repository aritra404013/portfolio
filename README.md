# Ares Web Solutions - Space Portfolio

Welcome to my full stack Next.js 14 space portfolio. This project features a stunning 3D design with interactive background stars and modern animations.

## 🚀 Features

- **3D Interactive Background**: Built with React Three Fiber and Three.js.
- **Modern UI**: Styled with Tailwind CSS and Framer Motion for smooth transitions.
- **Responsive Design**: Fully optimized for mobile and desktop devices.
- **Project Showcases**: Display of my latest web and software development projects.

## 🛠️ Tech Stack

- **Frontend**: Next.js 14, React, Tailwind CSS
- **3D Library**: Three.js, @react-three/fiber, @react-three/drei
- **Animations**: Framer Motion
- **Icons**: Heroicons, React Icons

## 🏁 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
